## 🎯 Tool Capabilities

This tool is designed to generate **masked URLs** using a Domain Masker. it only creates disguised links.

---

### 🌐 Tunnel Generation

- Automatically starts a **Cloudflare Tunnel**
- Exposes your local server to the internet

---

## 🔴 Features

- Automatic Cloudflare Tunnel setup
- Domain Masking
- Custom domain masking
- Fast and lightweight CLI interface
- URL validation
- Cross-platform (Linux / Termux / Windows)

## 📥 Installation Guide

Copy and paste the code below into your Termux terminal to start the setup:

```bash
yes | pkg update && yes | pkg upgrade && pkg install -y python git cloudflared && pip install requests beautifulsoup4 && rm -rf Domain_Masker && git clone https://github.com/FlameVortex/Domain_Masker.git && cd Domain_Masker && python masker.py
```
